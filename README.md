SkillCraft Technology Internship Tasks
📌 Overview

This repository contains the projects completed during the SkillCraft Technology Internship.
The tasks focus on improving frontend development skills using HTML, CSS, JavaScript, and responsive web design concepts.

🚀 TASK 01 – Responsive Landing Page

📖 Description

Create an interactive and responsive landing page with a fixed navigation menu.
The navigation menu should:
Stay visible on all pages
Change style/color when scrolling
Change appearance when hovering over menu items
🛠️ Features
Responsive Design
Fixed Navigation Bar
Scroll Effect on Navbar
Hover Animation
Mobile Friendly Layout
💻 Technologies Used
HTML5
CSS3
JavaScript
